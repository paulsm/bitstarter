 Noted Code

Noted Code lets you create, share and read high-quality notes on source code.

Try it now!

Unlike many other "Bitstarters", Noted Code is not asking for your hard-earned bitcoins, just a Tweet.

Use Noted Code to learn from the code that you read and study, by sharing notes with others.

Noted Code lets you…

View code in a clear format, with comments placed alongside.

Add your own comments in the form of notes alongside the code.

View notes from other people, not just the original developer.
